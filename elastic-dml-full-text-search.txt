#### Match query
Quels sont les documents qui contiennent le terme draw dans le champ app_name ?

// tag::dml-fts-match-query[]
GET tp_elastic_mq/_search
{
  "query":
  {
    "match": {
      "app_name": "draw"
    }
  }
}
// end::dml-fts-match-query[]
